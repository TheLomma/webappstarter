# Web App Launcher v1.1

Deine persoenliche App-Startseite als PWA.

## Features
- 6 Themes (Light, Dark, Blue, Blue Dark, Green, Green Dark)
- Deutsch / Englisch
- Apps hinzufuegen und loeschen
- localStorage Speicherung
- PWA-faehig

## Lokale Entwicklung

```
npm install
npm start
```

## Deployment auf Vercel

1. Repo auf GitHub pushen
2. vercel.com -> New Project -> GitHub Repo auswaehlen
3. Framework: Create React App (wird automatisch erkannt)
4. Deploy klicken
